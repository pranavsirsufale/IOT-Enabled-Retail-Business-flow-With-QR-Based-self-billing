# IOT-Enabled-Retail-Business-Flow-With-QR-Based-self-billing
This project is a Smart Retail Billing System based on IoT and QR code technology.
It allows customers to scan QR codes of products, automatically fetch product details from a database, and generate a digital bill without waiting at the billing counter.

The system reduces manual work, saves time, and improves the overall shopping experience.

---
## Objectives

Reduce long queues at billing counters

Enable self-billing using QR scanning

Maintain real-time product and inventory data

Automate bill generation

Improve retail efficiency using IoT concepts

---
## How the System Works

Each product has a unique QR code

Customer scans the QR code using the system

Product ID is matched with the database

Product details (name, price, category) are fetched

Bill is generated automatically

Final bill can be viewed, printed, or saved

## System Architecture

Frontend: React + Tailwind CSS

Backend: Django

Database: MySQL

QR Scanner: Camera / IoT device

Billing Logic: Automatic calculation
